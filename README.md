# DarkLearners_PS6

A Python-based **mobile forensics analysis tool** built using **PyQt6**.  
Designed for **non-root Android devices**, this tool performs forensic-safe
data extraction and analysis using ADB and open-source intelligence techniques.

---

##  Features

-  Android data acquisition via **ADB**
-  Image extraction based on **date range**
-  EXIF GPS metadata extraction
-  Interactive **Geo-location mapping**
-  Inbuilt forensic analysis modules
-  Evidence-safe processing (read-only pull)
-  Modular design (tab-based analysis)
